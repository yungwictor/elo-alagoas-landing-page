# Elo Alagoas Landing Page

Landing page premium da Elo Alagoas para captação de clientes nas frentes de loteamentos e aluguel de casas de temporada no litoral de Alagoas.

## Visão geral

O projeto foi construído como uma página estática de alta conversão, com identidade visual inspirada em resort de luxo e proptech moderna.

## Estrutura

- `index.html`: página principal com HTML, Tailwind CSS via CDN, Lucide Icons e JavaScript leve.
- `logo-elo-alagoas.png`: logo oficial utilizada no header e no footer.
- `netlify.toml`: configuração simples de publicação para deploy estático.
- `.gitignore`: exclusão de arquivos locais do Netlify.

## Destaques da interface

- Hero section com split CTA para loteamentos e temporada.
- Navegação flutuante em estilo glassmorphism.
- Busca rápida com tabs e envio direto para WhatsApp.
- Cards premium para loteamentos e casas de temporada.
- Animações suaves de entrada e microinterações.
- Layout totalmente responsivo.

## Deploy

O projeto está preparado para publicação estática em plataformas como Netlify.
O repositório público está com GitHub Pages habilitado a partir da branch `main`, permitindo deploy automático a cada novo push.
